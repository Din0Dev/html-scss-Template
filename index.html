This is Template HTML + SCSS With Reponsive

How to Use

Reponsive

  • Use class name in common
    eg: Row,Col-2,Col-3,...
  • Can edit grid to 12,24,32 in layout.scss
  • We have 4 options for custom reponsive in class: LARGE(1200px),MEDIUM(992px),SMALL(768px),XSMALL(576px)

Function Base

  • We have Function Base in Mixin
  • Create with @mixin animation(props)
  • Use call @include animation(props)

Color

  • Set up Color with variables
  • Create with @color: #000000
  • Use call background: $color

New CSS

  • Create file in folder SCSS/components (example.scss)
  • Import to app.scss
    (@import(SCSS/components/example.scss))

Run Live SCSS

  • Run in terminal npm run sass-compile
  • Import to app.scss
    (@import(SCSS/components/example.scss))

Use Font Size

  • Please custom PX to REM
  • eg: font-size: 10px => font-size: 1rem

Folder

  • app.scss: Import file in Folder SCSS
  • Folder abtracts
  • breakpoint: Setup PX for Reponsive
    mixins: Setup function css
    variables: Setup color,fontsize,width,height default
  • Folder components
  • Setup css for page,components
  • Folder layout
  • Setup grid,col for Reponsive
